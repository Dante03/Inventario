Requisitos:
1.- Entity Framework 6.1.3
2.- Net Framework 4.7.2

Crear la base de datos siguiendo los pasos a continuacion:
1.- Herramienas->Administrador de paquetes NuGet->Consola del Administrador de paquetes.
2.- Ingresar el siguiente comando en la ventana de la terminal que se muestra: update-database.
3.- Ejecutar el proyecto con IIS Express desde Visual Studio o presionar la tecla F5 para compilar y ejecutar.
